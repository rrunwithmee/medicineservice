# MedTime - Онлайн запись к врачу

## Описание проекта
Проект медицинской записи онлайн на React, который позволяет пациентам записываться к врачам различных специальностей, просматривать доступные даты и время приема, а также использовать лист ожидания.

## Основные функции
- Поиск врачей по специальности
- Выбор поликлиник и адресов
- Просмотр профилей врачей с фотографиями и квалификацией
- Онлайн запись на прием
- Лист ожидания на случай отмен
- Авторизация и личный кабинет
- Адаптивный дизайн для всех устройств

## Технологии
- **Frontend**: React, Vite, JavaScript
- **Стилизация**: CSS Modules, React Icons
- **Маршрутизация**: React Router DOM
- **Хранение данных**: LocalStorage
